# ⚙️ Herramientas utilizadas

   * Expo
   * React native
   * Expo-network
   * SQLite

# 🚀 Levantar Proyecto

```bash
    git clone -b master https://github.com/DiLucaLucas/app-crudo-check.git
```

```bash
   cd app-crudo-check
```

```bash
   npx install
```

```bash
   npx expo start -c
```
